ATIVIDADE 10 - OPERAÇÕES COM VETORES
====================================

Esta atividade contém 18 questões sobre operações com vetores em Java.
Cada questão está em um arquivo separado (Questao1.java até Questao18.java).

FUNÇÕES UTILIZADAS (conforme ensinado em sala):
- Estruturas de repetição (for)
- Operadores aritméticos e lógicos
- Vetores (arrays)
- Funções básicas sem importação de bibliotecas

CONTEÚDO DAS QUESTÕES:
=======================

1. Questao1.java - Soma de elementos
   Calcula a soma de todos os elementos de um vetor.

2. Questao2.java - Média de elementos
   Calcula a média aritmética dos elementos.

3. Questao3.java - Maior valor
   Encontra o maior número no vetor.

4. Questao4.java - Menor valor
   Encontra o menor número no vetor.

5. Questao5.java - Contar pares
   Conta quantos números pares existem no vetor.

6. Questao6.java - Contar ímpares
   Conta quantos números ímpares existem no vetor.

7. Questao7.java - Positivos e negativos
   Conta a quantidade de números positivos e negativos.

8. Questao8.java - Inverter lista
   Cria um novo vetor com elementos em ordem inversa.

9. Questao9.java - Verificar palíndromo
   Verifica se o vetor é igual quando lido de trás para frente.

10. Questao10.java - Multiplicar elementos
    Multiplica todos os elementos do vetor por 2.

11. Questao11.java - Remover duplicatas
    Cria um novo vetor sem elementos repetidos.

12. Questao12.java - Segundo maior valor
    Encontra o segundo maior número no vetor.

13. Questao13.java - Ordenar vetores
    Implementa o algoritmo Bubble Sort para ordenação crescente.

14. Questao14.java - Busca linear
    Verifica se um elemento existe no vetor.

15. Questao15.java - Contar ocorrências
    Conta quantas vezes um valor aparece no vetor.

16. Questao16.java - Produto escalar
    Calcula o produto escalar entre dois vetores.

17. Questao17.java - Interseção de vetores
    Encontra os elementos comuns entre dois vetores.

18. Questao18.java - União de vetores
    Combina dois vetores sem repetições.

COMO COMPILAR:
===============
No diretório /home/matheus/Documentos/GitHub/POO/atvd10/src, execute:

javac Questao1.java
javac Questao2.java
... e assim por diante

Ou compile todas de uma vez:
javac *.java

COMO EXECUTAR:
===============
Após compilar, execute cada uma com:

java Questao1
java Questao2
... e assim por diante

Cada execução gerará um vetor aleatório e exibirá o resultado da operação.

CARACTERÍSTICAS:
=================
- Todos os programas geram vetores aleatórios automaticamente
- Usam apenas estruturas básicas ensinadas em sala de aula
- Sem uso de bibliotecas externas
- Código comentado e legível
- Exemplos de saída em cada arquivo
